Generate the alpha (annotation density), gamma1 (FNR), gamma2 (FPR) parameters from the data.
(python 2.7)

1. Run the scripts in the IAA directory first.
2. Update the file params_from_data_config.txt to have the correct full paths to the corpus files.
3. python ECO-CollecTF_data_parameters.py

See the end of the files:
1. corpus_alpha_from_data_log.txt 
2. corpus_FNR_FPR.txt
